    Relief

    为二分类问题设计。

    2.论文

  1. Stratified Feature Ranking（SFR）：分层特征排序
  2. SFC 即(Subspace Feature Clustering) : 子空间特征聚类
  3. SWCC：全称Subspace Weighting Co-Clustering。子空间加权共同聚类
    在SFR中，首先提出SFC来同时识别特征聚类和每个类别的每个特征的重要性。SFC通过消费类标签来扩展子空间加权共同聚类 (SWCC) 。利用 SFC 的联合聚类结果，根据 SFC 学习的子空间权重，分别对不同特征聚类中的特征分别进行排序。
