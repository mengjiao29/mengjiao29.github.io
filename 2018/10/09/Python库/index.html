#1.os库 #
　　用于对文件目录的处理

1.1读取文件

　　此方法在读取文件出错时，可以关闭文件。

with open('/path/to/file', 'r') as f:
data = f.read()

  • r:读；

  • w:写;

  • a:追加;

  • b:二进制模式;

  • +:读写模式;
    ###1.2 os.getcwd() ###
    　　显示当前文档的路径

    1.3 os.chdir(PATH)

    　　用于改变当前工作目录到指定的路径。
