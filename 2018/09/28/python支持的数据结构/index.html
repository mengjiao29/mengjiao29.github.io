　　Python支持容器（container）。主要的容器：序列、映射和集合。

1.序列

　　序列中每个元素都是有编号的，即其位置或索引。序列可以包括其他序列。常用的序列有：列表、元组和字符串。

　　通用序列操作：索引、切片（[a:b:c:]）、相加（加法+）、相乘（*）、迭代和成员资格检查（in）

1.1列表

　　列表是可以修改的，适用于需要中途添加元素。

　　表示方法：所有元素放入方括号内，并用逗号隔开。
比如：

>>>a = ['Minny','32']
>>>b = ['Jonhn','25']
>>>database = [a,b]
>>>database
[['Minny,'32'],['Jonhn','25']]

1.2元组

　　元组是不可修改的序列，保证数据安全。

　　表示方法：用圆括号括起，值用逗号分隔开。

>>>(1,2,3)
(1,2,3)

　　只包含一个值的元组，必须在值后面加上逗号，否则为数组或出错。空元组为：()。

1.3字符串

　　字符串是不可变的，所有元素赋值和切片赋值都是非法的。

2.映射

　主要的映射是字典。字典是通过花括号{}或dict函数来构造键-值对的。字典查找速度快。

　表示方法：用花括号括起，键与值用冒号分隔，项之间用逗号隔开。

{'name':'Minny','age':'25'}

　常用操作：

  • 创建：dict()

    >>>items=[('name,'Minny'),('age','25')]
>>>data=dict(items)
>>>data
{'name':'Minny','age':'25'}
>>>data['name']
Minny

  • 删除键为k的项：del(k)

  • 检查字典d是否包含键为k的项：k in d

  • 访问字典项：get

　　a.访问不存在的键时，不报错，返回None

>>>data.get('lala')
None

　　b.可指定返回值，这样不不存在时返回指定值而不是none。

>>>data.get('lala', 'N/A')  
N/A
